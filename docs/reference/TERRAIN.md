### `Mesh.MakeHeightmap(imageHandle, width#, height#, length#)`

Creates a terrain mesh from a heightmap image.

- `imageHandle`: Handle to an image where color brightness determines height.
- `width#`, `height#`, `length#`: The dimensions of the resulting mesh in 3D space.
