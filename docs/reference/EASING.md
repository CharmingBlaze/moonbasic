# Easing (`EASEIN`, `EASEOUT`, `EASELERP`, …)

Easing and interpolation helpers are registered from **`runtime/mbgame/register_ease_noise_rand.go`** (and related). Inputs are typically **normalized times** or **positions** in **0..1** or world units depending on the function.

See the manifest and **`docs/API_CONSISTENCY.md`** for the full list of **`EASE*`** keys.
