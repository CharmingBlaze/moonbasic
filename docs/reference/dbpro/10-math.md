# DBPro — Math

moonBASIC uses **`SIN`**, **`COS`**, … and **`MATH.*`** overloads — [MATH.md](../MATH.md).

| DBPro | moonBASIC | Notes |
|-------|-----------|--------|
| **SIN** / **COS** / **TAN** | ✓ **`SIN`**, **`COS`**, **`TAN`** | Also **`MATH.SIN`**, … |
| **ASIN** / **ACOS** / **ATAN** | ✓ **`ASIN`**, **`ACOS`**, **`ATAN`** | |
| **SQRT** | ✓ **`SQRT`** | |
| **ABS** | ✓ **`ABS`** | |
| **RND (low#, high#)** | ✓ **`RND`** | See arity in manifest. |
| **RANDOMIZE (seed)** | ✓ **`RANDOMIZE`**, **`RNDSEED`** | |

**Note:** DBPro **SQR** sometimes meant “square” in other BASICs; moon **`SQR`** is documented as **square** in some manifests — use **`SQRT`** for √x.
