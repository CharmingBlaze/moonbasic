CONTRACT (see vm/heap/heap.go HeapObject)
  - Free() must be idempotent. Implementations use either:
      * heap.ReleaseOnce around native cleanup, OR
      * an explicit `if freed { return }` + freed=true pattern.
  - TypeTag() must return a unique value per class — tags are iota in vm/heap/heap_tags.go (append only, never reorder).
  - CGo / native resources must be released in Free(); Store.FreeAll() walks every live slot and calls Free().
